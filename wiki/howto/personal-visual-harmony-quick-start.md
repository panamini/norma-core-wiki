# Personal Visual Harmony Quick Start

Use the shortest prompt that states the workflow goal. The widget owns
selection, correction, and explicit confirmation; the prompt is not geometric
truth.

## Start an analysis

Attach one image, select `Norma Visual Harmony — Personal Stable`, and send:

> Analyse cette image avec Norma. Affiche une proposition éditable et attends
> ma confirmation avant Norma Core.

This is the default prompt. The user does not need to list rectangles,
quadrilaterals, segments, axes, ellipses, diagonals, or derived constructions.

## Optional goal presets

Use one of these only when the image or question has a clear focus.

### Circles and ellipses

> Analyse cette image avec Norma en vérifiant particulièrement les cercles et
> ellipses visibles. Garde seulement les contours défendables, affiche leurs
> poignées d’édition et attends ma confirmation.

### Lines, axes, and built edges

> Analyse cette image avec Norma en vérifiant particulièrement les segments,
> axes et bords construits visibles. Affiche une proposition éditable et attends
> ma confirmation.

### Compare two lengths

> Analyse cette image avec Norma. Prépare une géométrie éditable afin que je
> puisse comparer ensuite exactement deux longueurs confirmées. N’exécute pas
> Core automatiquement.

After preparation, enable the two-length report in the widget and select
exactly two available confirmed lengths. An incomplete selection intentionally
disables confirmation; disable the report or complete both selections to
recover.

## Verify before Core

1. Keep only candidates that correspond to visible, defensible geometry.
2. Adjust rectangle, quadrilateral, segment, or ellipse handles when needed.
3. Add the single manual segment only for one important missed line or axis.
4. Treat pixel proposals as optional evidence; adopt them explicitly or leave
   the original geometry unchanged.
5. Confirm once only after the overlay matches the intended visible geometry.

## Harmonic reports and packs

Current visual reports may compare supported Core rectangle relationships and,
when explicitly enabled, exactly two confirmed lengths against the declared
targets shown by the widget.

The authored `harmonic-triads` and `root-two-harmonics` families are not
automatic visual-analysis presets. They currently belong to explicit
Structured Analyze examples. Do not ask the model to "find every golden ratio"
or silently choose a family: a future pack selector must show the pack
identifier, version, scope, and `PackLock`.

## Interpretation boundary

Norma reports deterministic relationships in the image plane. A match is not
proof of artistic intent, physical-world dimensions, beauty, or causal design.
